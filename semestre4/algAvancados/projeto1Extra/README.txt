# Siga as intruções a seguir para compilar e executar o programa

1. UTILIZAÇÃO COM g++ ou gcc (supondo que o usuário já possua um dos dois programas instalados em sua máquina)
	
    - Abra um terminal na pasta gerada pela extração do .zip
    - Para compilar, digite no terminal o comando: g++ orcamentoExtra.cpp -o <nomeDoExecutavel> ou gcc orcamentoExtra.cpp -o <nomeDoExecutavel>
    - Substitua <nomeDoExecutavel> pelo nome que desejar chamar o arquivo executável
    - Para executar o programa digite no terminal:./<nomeDoExecutavel>     
    - Digite os dados de entrada do programa como especificado no pdf do projeto
