SAIBA MAIS SOBRE O SITE

O wwwFIT foi desenvolvido com o intuito de fornecer atividades físicas para os usuários em busca de uma forma corporal ideal. Visando sempre a saúde do cliente, o nosso aplicativo fornece o tempo necessário - e ideal - de conclusão de cada exercício.

Desenvolvedores:

  • Eduardo Pennone;
  • Henrique dos Santos;
  • Leonardo Rossi;
  • Witor de Oliveira.

A aplicação foi desenvolvida utilizando um banco de dados (MySQL), o framework Spring Tools do JAVA - responsável pela conexão entre o frontend (HTML/CSS) e o backend (JAVA) - e suas dependências (thymeleaf, Spring MVC, Security Boot e outras).

Universidade de São Paulo (USP) - São Carlos 2019

Professor Dr. Marcio Delamaro
